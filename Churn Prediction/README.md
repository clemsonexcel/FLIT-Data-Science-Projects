# Churn Prediction 

## Project Overview
This project aims to develop a predictive model to identify potential customer churn for a telecommunications company. Using customer data, the model predicts whether a customer is likely to churn based on various service attributes and customer demographics.

## Tools and Technologies
- **Python**: Primary programming language used.
- **Pandas**: Library for data manipulation and analysis.
- **Scikit-Learn**: Used for modeling and predictions.
- **Streamlit**: For creating the web application.
- **Joblib**: For model serialization and deserialization.


## Model
The project utilizes a logistic regression model due to its efficacy in binary classification tasks. The model is trained with cross-validation to optimize hyperparameters and validate its performance.

## Deployment
The model is deployed as a web application using Streamlit, which allows users to interactively input customer data and receive churn predictions.
